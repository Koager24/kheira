# Optionale Premium-Embleme (Higgsfield / Nano Banana)

Die App zeigt standardmäßig die eingebauten Vektor-Wappen. Liegen hier PNGs,
werden sie automatisch als Emblem-Kern verwendet (Fallback bleibt das SVG):

  konsistenz.png    – Doppelwelle (Kategorie "Regelmäßigkeit")
  heilungsweg.png   – aufsteigender Pfad zur Sonne
  volle-tage.png    – geschlossener Ring mit Haken über Welle

Empfehlung: quadratisch, weißer Hintergrund, 512–1024 px.
